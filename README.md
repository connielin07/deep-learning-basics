# Deep Learning Basics

This repository contains beginner-level artificial intelligence and deep learning practice notebooks.  
It covers perceptron-based logic gates, multilayer perceptron prediction, and convolutional neural network basics.

本專案整理人工智慧概論與深度學習基礎練習，內容包含感知器、MLP 多層感知器與 CNN 卷積神經網路等入門主題。

## Project Purpose

The purpose of this repository is to record and organize my introductory deep learning practice.  
It focuses on understanding the basic structure and workflow of neural networks rather than building a production-level AI system.
